# 📇 Contact Management System (Java)

A simple **Java-based Contact Management System** that allows users to **add, update, search, delete, and display contacts**. This project demonstrates core Java concepts like **OOP (Object-Oriented Programming)**, **classes**, **constructors**, and **ArrayList**.

---

## 🚀 Features

* ➕ Add new contacts (Name, Phone, Email)
* 🔍 Search contacts by name
* ✏️ Update existing contact details
* ❌ Delete contacts
* 📋 Display all saved contacts
* 🧠 Case-insensitive search

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Classes, Objects, Encapsulation)
* ArrayList (Dynamic Data Structure)
* Scanner (User Input)

---

## 📁 Project Structure

```
Contact-Management-System/
│
├── Contact.java          # Contact model class
├── ContactManager.java  # Handles operations (CRUD)
└── Main.java            # User interface (menu-driven)
```

---

## ▶️ How to Run

1. **Clone the repository**

```bash
git clone https://github.com/your-username/contact-management-system.git
```

2. **Navigate to the project folder**

```bash
cd contact-management-system
```

3. **Compile the Java files**

```bash
javac Main.java
```

4. **Run the program**

```bash
java Main
```
